# Employee Compensation Analysis

This project involves analyzing employee compensation data to identify pay gaps and equity.

### Tools Used:
* **Excel:** Data cleaning and pivot tables.
* **Tableau:** Visualizing the final insights.

### Key Insights:
* Found that more than **50% of employees** were in the "Unfairly and Highly Discriminatively Compensated" category.
* Identified specific departments with the highest pay discrimination.
